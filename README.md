# 4.Хитрая функция

**Условие:**  
Функция 𝑓(𝑛) определена следующим образом: запишем все цифры десятичной
записи числа 𝑛 в обратном порядке, после чего избавимся от лидирующих нулей.
Примеры:
● 𝑓(221) = 122
● 𝑓(1020) = 201
● 𝑓(10) = 1
Определим функцию 𝑔(𝑛) = 𝑓(𝑓(𝑛)) / 𝑛 .
Примеры:
● 𝑔(221) = 1
● 𝑔(1020) = 0. 1
● 𝑔(10) = 0. 1
